# Traffic Management System Simulation (OpenGL)

This project is a 2D Traffic Management System simulation developed using OpenGL and GLUT in C. The program simulates a four-way traffic intersection where vehicles move according to traffic signals.

## Features

* Four-way traffic intersection
* Traffic signals with Red, Yellow, Green lights
* Cars move in four directions
* Cars stop at red signal and move at green signal
* Smooth acceleration and deceleration
* Timer-based signal control
* Continuous animation using GLUT

## Technologies Used

* C Programming
* OpenGL
* GLUT

## How to Run

Compile using:

```
gcc traffic.c -lGL -lGLU -lglut -lm
./a.out
```

## Project Output

The program shows a graphical simulation of a traffic intersection with working traffic signals and moving vehicles.
